# AI Meeting Follow-Up Workflow (n8n)

This workflow extracts action items from meeting transcripts and distributes them automatically.

Flow:

Meeting transcript  
→ AI extracts tasks and summary  
→ Slack sends individual to-dos  
→ Email sends meeting summary  

Built with n8n while exploring workflow automation.
